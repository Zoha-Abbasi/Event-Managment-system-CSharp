Event Management System
📌 Overview

The Event Management System is a software application designed to help hotels and marquees manage event bookings efficiently. It allows venue owners to register, log in, manage customer data, handle bookings, and generate receipts through an interactive dashboard.

✨ Features
🔐 User Registration and Login system
🏨 Hotel / Marquee selection with dynamic input forms
📊 Dashboard for managing operations
👥 Customer data management (Add, Update, Delete)
--Automatic payment calculation according to type of  hall /room choosed
🔎 Search and filter customer records
🛏️ Room booking and event management
🧾 Receipt generation and printing
💾 Database integration for secure data storage
-- CRUD operations
-- Update/delete user record


⚙️ System Workflow
User signs up and logs into the system
Selects venue type (Hotel or Marquee/both)
Enters required venue details
Add business info
Accesses dashboard
Genrate monthly and yearly Reports
-- Performs operations:
Add customer data
Manage bookings (rooms/events)
Search and filter records
make room/marquee booking
print payment reciept
Update or delete data
Generate receipts
View latest bookings


🛠️ Technologies Used
Programming Language: C#
Framework: ASP.NET / Windows Forms
Database: SQL Server
Frontend: Basic UI Components (Forms)
Backend: Database connectivity using SQL queries
🗄️ Database Structure

The system uses a relational database with the following main tables:

Users – Stores login credentials
Venues – Stores hotel or marquee information
Customers – Stores client/event data
Bookings – Stores room and event booking details
